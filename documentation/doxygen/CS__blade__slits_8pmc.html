Detailed Description

Define motion for 2 slit blades

Original Author: Tom Cobb

Defined axes:

  • X (addr 6) = centre in same EGUs as blade
  • Y (addr 7) = gap in same EGUs as blade

Macros:

  • COORD = CS number, e.g. 2
  • PLC = PLC number, should be CS number+15, e.g. 17
  • BP = Axisnum for Blade+, e.g. 1
  • BM = Axisnum for Blade-, e.g. 3
