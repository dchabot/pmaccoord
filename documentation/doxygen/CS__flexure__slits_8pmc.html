Detailed Description

Define motion for flexure slits

Original Author: Alan Greer

Defined axes:

  • X (addr 6) = gap in same EGUs as actuator

Macros:

  • COORD = CS number, e.g. 2
  • PLC = PLC number, should be CS number+15, e.g. 17
  • AS = Axisnum for actuator stroke, e.g. 1
  • LL = Lever length in mm
  • SW = Slit width at U=0 in mm
