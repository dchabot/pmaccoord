Detailed Description

Define motion for Energy in Accel DCM

Original Author: Tom Cobb. Modified by Emma Shepherd to add support for backlash/hysterisis correction (see PROG_CS_motion_backlash.pmc)

Defined axes:

  • X (addr 6) = Energy in kEv
  • Y (addr 7) = Offset in mm

Macros (and example values):

  • COORD = CS number (only works for CS 1..9), e.g. 2
  • PLC = PLC number, should be CS number+15, e.g. 17
  • BRAGG = Axisnum for Bragg, e.g. 2
  • T2 = Axisnum for T2, e.g. 3
